## 4. 인증 규칙

### 4.1 고객 앱 인증
| 규칙 ID | 규칙 | 동작 |
|---|---|---|
| AUTH-C01 | 토큰 없으면 /setup 이동 | 자동 리다이렉트 |
| AUTH-C02 | 토큰 검증 실패 시 재설정 | localStorage 클리어 + /setup 이동 |
| AUTH-C03 | 새로고침 시 토큰 유지 | localStorage에서 복원 |

### 4.2 관리자 앱 인증
| 규칙 ID | 규칙 | 동작 |
|---|---|---|
| AUTH-A01 | 토큰 없으면 /login 이동 | 자동 리다이렉트 |
| AUTH-A02 | 16시간 토큰 만료 | API 401 감지 → 자동 로그아웃 |
| AUTH-A03 | 5회 로그인 실패 시 차단 | 15분 차단 메시지 표시 (서버 응답 기반) |
| AUTH-A04 | 역할 기반 메뉴 표시 | hq_admin만 "매장 관리" 메뉴 표시 |
| AUTH-A05 | 새로고침 시 토큰 유지 | localStorage에서 복원 |

## 5. UI 인터랙션 규칙

### 5.1 터치 인터페이스 (고객 앱)
| 규칙 | 값 | 근거 |
|---|---|---|
| 최소 터치 타겟 | 44x44px | WCAG 2.1 Success Criterion 2.5.5 |
| 버튼 간 최소 간격 | 8px | 오터치 방지 |
| 스크롤 방향 | 세로 스크롤 기본 | 태블릿 자연스러운 인터랙션 |
| 카테고리 탭 | 가로 스크롤 | 카테고리 수 많을 때 대응 |

### 5.2 모달 규칙
| 규칙 | 동작 |
|---|---|
| 배경 클릭 시 닫기 | 일반 모달은 닫기, 확인 모달은 닫지 않음 |
| ESC 키 닫기 | 모든 모달 닫기 |
| 스크롤 잠금 | 모달 열릴 때 배경 스크롤 비활성화 |

### 5.3 로딩 상태
| 상황 | 표시 |
|---|---|
| 페이지 초기 로드 | 전체 화면 로딩 스피너 |
| API 호출 중 | 버튼 로딩 상태 (스피너 + 비활성화) |
| 폴링 중 | 별도 표시 없음 (백그라운드) |
| 이미지 로드 | 스켈레톤 플레이스홀더 |

### 5.4 에러 처리 패턴
| 에러 유형 | 표시 방식 | 재시도 |
|---|---|---|
| 네트워크 에러 | 토스트 메시지 | 자동 재시도 (폴링) 또는 수동 |
| 유효성 검증 에러 | 필드 하단 인라인 에러 | 즉시 수정 가능 |
| 서버 에러 (500) | 에러 메시지 + 재시도 버튼 | 수동 재시도 |
| 인증 에러 (401) | 자동 로그아웃 + 리다이렉트 | 재로그인 |
| 충돌 에러 (409) | 인라인 에러 메시지 | 값 수정 후 재시도 |

## 7. 반응형 브레이크포인트

| 브레이크포인트 | 범위 | 대상 |
|---|---|---|
| `sm` | 640px~ | 소형 태블릿 |
| `md` | 768px~ | 일반 태블릿 (세로) |
| `lg` | 1024px~ | 일반 태블릿 (가로), iPad |
| `xl` | 1280px~ | 대형 태블릿, PC |

**고객 앱**: md~lg 최적화 (태블릿 세로/가로)
**관리자 앱**: lg~xl 최적화 (PC/대형 태블릿)
